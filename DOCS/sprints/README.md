## Como Preencher

Cada pasta de sprint deve concentrar:

- Objetivo da sprint.
- Período.
- Backlog comprometido.
- Tarefas executadas.
- Evidências.
- Link para vídeo ou demonstração.
- Riscos, impedimentos e aprendizados.

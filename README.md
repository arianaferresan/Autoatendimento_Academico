# FAQtec

Projeto acadêmico desenvolvido para a Fatec Jacareí com o objetivo de apoiar a Secretaria Acadêmica por meio de uma plataforma de autoatendimento.

## Minha atuação

Atuei principalmente no desenvolvimento Frontend e na experiência do usuário (UX/UI), com foco nas áreas administrativas do sistema.

Principais contribuições:

* Desenvolvimento e evolução das telas administrativas
* Melhorias de usabilidade e experiência do usuário
* Refinamento visual da interface
* Organização visual de formulários, tabelas e dashboards
* Melhorias de navegação e fluxo entre páginas
* Correções visuais e otimização da experiência administrativa
* Participação na modelagem e documentação do sistema
* Elaboração e manutenção de diagramas UML

## Tecnologias utilizadas

* React
* TypeScript
* Vite
* Tailwind CSS
* Git e GitHub

## Destaques

Durante o projeto participei ativamente da construção da área administrativa, buscando tornar a experiência mais intuitiva, organizada e eficiente para os usuários internos da instituição.

O foco das melhorias esteve na clareza das informações, consistência visual e facilidade de utilização dos recursos administrativos.

## Aprendizados

Este projeto proporcionou experiência prática em:

* Desenvolvimento Frontend
* UX/UI Design
* Trabalho em equipe utilizando Git
* Metodologias ágeis
* Levantamento e implementação de requisitos
